HTML entity문자(키보드문자), 심볼문자

목록

1. entity문자, 심볼문자

html에서 엔티니 문자(키보드 문자, html 문자)나 심볼문자를 나타내는 방법은 다음과 같다.
  1. &entity_name;
  2. OR
  3. &#entity_number;

2. 스페이스 키

웹페이지에서 스페이스키를 공백은 한칸밖에 인식하지 않는다. 만약 여러칸의 공백을 입력하기 위해서는 " " 엔티티
문자로 공백을 나타내야 한다.

3. html문자

"<",">"문자를 웹페이지에 표현하기 위해서는 < , >를 사용하여야 한다. 키보드의 <,>로 표현하면 태그로 인식한다.

I will display       .

I will display <

I will display >

4. 심볼문자(링크를 참조)

  • 수학기호 심볼 문자 : http://www.w3schools.com/charsets/ref_utf_math.asp
  • 그리스기호 심볼 문자 : https://www.w3schools.com/charsets/ref_utf_greek.asp
  • 통화기호 심볼 문자 : https://www.w3schools.com/charsets/ref_utf_currency.asp
  • 화살표 심볼 문자 : https://www.w3schools.com/charsets/ref_utf_arrows.asp
  • 기타(별,하트,..) 심볼 문자 : https://www.w3schools.com/charsets/ref_utf_symbols.asp
